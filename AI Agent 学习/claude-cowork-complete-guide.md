# Claude Cowork 完整使用指南：从设置到实战拆解

> Claude Cowork 真的完全不输 OpenClaw，但是很多人还不知道。它是 Anthropic 年初推出的桌面端 Agent，能让 AI 直接操作本地文件、自动完成多步骤任务。简单来说，它把 Claude 从聊天机器人变成了能帮你干活的助理。

Nav Toor 是个品牌策略师，分享了他如何用 Claude Cowork 搭建自己的工作流，现在写客户提案、整理会议纪要、做数据分析报告这些活儿，**20 分钟就能完成以前 3 小时的工作量**。

---

## 普通 AI 工具的痛点：我们一直在来回搬运

大多数人是这样用 AI 的：

- 打开浏览器，进入 ChatGPT
- 把本地文件的内容复制粘贴进去
- 等 AI 生成结果
- 复制回来，保存到本地

这个"复制粘贴"的循环每天重复十几次，光在这上面就浪费一个多小时。这种模式让 AI 始终是个外部工具，没法真正融入你的工作流。

Nav 在过去三年试了上千个提示词、几十个工具后，发现了一个关键洞察：**与其不断优化提示词，不如优化 AI 能访问的上下文。**

## Cowork 到底强在哪：5 个核心功能

### 1. 直接操作本地文件

普通 AI 在浏览器里，你要复制粘贴。Cowork 装在你电脑上，直接打开你的文件夹读文件、写文件。

**具体能做什么：**
- 你的旧报告、品牌手册，它都能直接参考
- 生成的文档自动存到指定位置，不用手动整理
- 可以在任务中途引用你的格式模板，而你不用特意告诉它

**怎么设置：**
1. 去 claude.com/download 下载桌面应用（支持 macOS 和 Windows）
2. 需要付费订阅：Pro 版 $20/月，Max 版 $100-200/月（重度使用）
3. 打开应用，顶部切换到 Cowork 标签
4. 点击"Select Folder"选择你电脑上的一个文件夹
5. 该文件夹里的所有文件，AI 都能直接读取

**Nav 的核心技巧：Context 文件夹策略**

创建一个名为"Claude Context"的文件夹，里面放三个文件：

**about-me.md** - 介绍你自己：
- 你是谁，做什么工作
- 你的角色和职责
- 什么样的工作成果算"好"
- 一个你引以为傲的工作案例

**brand-voice.md** - 你的沟通风格：
- 你常用的表达方式
- 哪些词你绝对不用
- 你觉得听起来不对的语句
- 几段你写得很好的文字示例

**working-style.md** - 你的工作偏好：
- 你喜欢 AI 先问问题再动手，还是直接执行？
- 你喜欢简短的回答还是详细的？
- 默认用什么文件格式（Word、Markdown、PDF）？

> 你给 AI 的上下文质量越高，你需要的提示词就越少。输出质量会从"像 AI 写的"变成"像你自己写的"。而且这几个文件会随时间复利，每周完善一点，AI 对你的理解就会更深。

**试试这个提示词：**
```
完整读取此文件夹中的所有文件。然后给我一个摘要：你了解我什么，我如何工作，以及你可以访问什么上下文。
```

### 2. 对模糊指令的判断

普通 AI 遇到模糊指令会怎么做？它会自信地给你一个看起来很完美但可能完全不对的答案。

而 Cowork 不同。当它需要更多信息时，会停下来生成一个表单问你：
- "你是想要 A 方案还是 B 方案？"
- "这个任务的重点是展示成果还是讨论问题？"
- "你需要包含数据图表吗？"

**每条提示里都用这个：**
```
我想要[你的任务]，以便[好的结果长什么样]。首先，在回复之前完整读取所有上传的文件。

还不要开始执行。问我澄清问题来完善方法。只有在我们对齐后才开始工作。
```

让 AI 自己搞清楚它需要知道什么。

### 3. 插件系统

Cowork 有 11 个官方插件，把通用 AI 能力封装成针对具体角色的工具：

| 插件 | 用途 |
|------|------|
| **Productivity** | 任务管理、日程安排 |
| **Marketing** | 内容创作、营销活动规划 |
| **Sales** | 客户研究、电话准备、外联邮件 |
| **Finance** | 财务建模、分析、报告 |
| **Data Analysis** | SQL、数据仪表板、异常检测 |
| **Legal** | 合同审查、法律研究 |
| **Product Management** | 产品规格、路线图、用户故事 |
| **Customer Support** | 工单处理、回复起草 |
| **Enterprise Search** | 跨工具搜索 |

**安装方法：**
1. 打开 Cowork，点击聊天栏的"+"按钮
2. 点"Plugins"浏览插件库
3. 选择适合你工作的插件，点击 Install
4. 在聊天框输入"/"查看该插件的斜杠命令

**使用示例：**
- 装了 Marketing 插件后：`/marketing: draft-content 写一篇关于【主题】的 LinkedIn 帖子，匹配我的 brand-voice.md 里的语气`
- 装了 Data 插件后：`/data: explore 分析这个文件夹里的 CSV 文件，标记异常，建议三个值得做的分析`

插件激活后的输出明显更有结构、更有主见——它知道你这个职能的好输出应该长什么样。

### 4. 全局指令：让 AI 记住你是谁

Cowork 有个让人抓狂的限制：每次新开对话，它都忘了你是谁。没有跨会话记忆。

**解决方案：Global Instructions（全局指令）**

这是你自己写的"记忆植入"，每次会话启动时自动加载。

**设置方法：**
1. 去 Settings > Cowork
2. 点击 Global Instructions 旁边的"Edit"
3. 写下你希望 AI 永远知道的信息

**里面写什么：**
- 你的名字、角色、工作内容（"我是做 DTC 品牌的策略师"）
- 沟通偏好（"我喜欢简洁直接的回答，不要废话"）
- 默认设置（"除非我另有说明，否则保存为 Word 文档"）
- 工作风格（"开始任务前先问澄清问题"）
- 禁忌事项（"除非我要求，否则不要用项目符号"）

**验证方法：**
```
在我们开始任何工作之前，告诉我你了解我什么，我喜欢如何工作，以及你知道的任何常驻偏好。
```

### 5. 连接器：打通你的工具链

Slack、Google Drive、Notion、Figma、Asana……Cowork 能连接 50 多个工具。

**授权一次后，你可以直接说：**
- "总结 #project-alpha 过去两周 Slack 的关键决策"
- "从 Drive 的收入文档里提取 Q1 数据"
- "找出 Notion 里标记为 blocker 的所有任务"

不用复制粘贴，AI 直接读取数据。

**连接方法：**
1. 去 Settings > Connectors
2. 浏览目录，找到你常用的工具
3. 点击"Add"，完成 OAuth 授权
4. 之后每次会话都能直接访问

> 这是所有套餐都免费的功能，也是 Cowork 里**最被低估的功能**。

## 实际操作实验

**任务**：准备明天客户会议的汇报材料

| 时间 | 发生了什么 |
|------|-----------|
| T+0 | 选择 Context 文件夹，输入任务 |
| T+30秒 | 系统弹出 3 个澄清问题（会议重点/输出格式/是否含图表） |
| T+2分钟 | 回答问题后，AI 输出详细执行计划 |
| T+3-18分钟 | 系统自动读文件、分析记录、生成文档、保存 |
| T+20分钟 | 用户回来验收，文档已完成 |

**关键洞察**：用户只参与了"定义需求"和"验收结果"两个环节，中间 15 分钟的执行过程完全自动化。

## 重要提醒：Cowork 的 6 个限制

| 限制 | 解决方法 |
|------|---------|
| 没有跨会话记忆 | 把重要信息写进 Context 文件 |
| 退出应用会中断任务 | 保持应用在后台运行（睡眠可以，退出不行） |
| 用量消耗比普通聊天快 | Settings > Usage 查看剩余额度 |
| 只有桌面端 | Context 文件夹放 iCloud/Dropbox 实现文件同步 |
| 不生成图片 | 配合 Gemini Imagen 或 Midjourney 使用 |
| 还是预览版 | 重要文件先备份，敏感操作前让 AI 先输出执行计划 |
